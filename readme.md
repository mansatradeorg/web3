# 30 mins skill test for Web3 developers

Wallectconnect is primary skill of senior Blockchain developers.
You have to do walletconnect on the project, then send result as video using loom.

The result must include:
- switching addresses
- switching chains
- swithcin wallets

### Built with
* React
* Tailwind CSS
* Node.js
* MongoDB


## Environment
node version > 16  
npm version > 8

## Installation
```bash
npm install
npm start
```
